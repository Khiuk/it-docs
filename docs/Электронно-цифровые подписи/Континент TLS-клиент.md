## Настройка соединения tls.ofukem.ru

Для настройки соединения необходимо:

1. Открыть континент TLS-клиент
2. Нажать "Добавить"
3. Указать следующие параметры

| Добавление ресурса |
| :----------------- | :-------------- |
| **Адрес:**         | `tls.ofukem.ru` |
| **Имя ресурса:**   | `tls.ofukem.ru` |
| **Порт ресурса:**  | `6005`          |
| **Тип:**           | `Туннель`       |
